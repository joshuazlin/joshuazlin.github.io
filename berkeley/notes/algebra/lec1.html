  • Let \(M_n(\mathbb{R})\) be the set of all \(n \times n\) matrices such that each entry \(a_{ij} \in \mathbb{R}\).
  • We can add two matrices \(A,B\) with entries \(a_{ij},b_{ij}\) to form a matrix with entries \((a_{ij}+b_{ij})\). We can multiply a matrix \(A\) by a constant \(\alpha\) to form a matrix with entries \(\alpha a_{ij}\). We can multiply two matrices \(A,B\) to form a matrix with entries \(\sum_k a_{ik}b_{kj}\).
  • For \(A,B \in M_n(\mathbb{R})\), \(A+B = B+A\), \(A(B+C) = AB + AC\), \(A(BC) = (AB)C\), but \(AB\) does not necessarily equal \(BA\). There is also a zero matrix denoted by \(0\), whose entries are all zero, such that \(A+0 = 0+A = A\). There is also an identity matrix denoted by \(I\), whose entries are all zero apart from the main diagonal that runs from the top left to the bottom right whose entries are one, such that \(AI = IA = A\).
  • A matrix \(A \in M_n(\mathbb{R})\) is invertible if and only if there exists a matrix \(A^{-1} \in M_n(\mathbb{R})\) such that \(AA^{-1} = A^{-1}A = I\). There is also a function called the determinant, \(\text{det}: M_n(\mathbb{R}) \rightarrow \mathbb{R}\) such that \(A \in M_n(\mathbb{R})\) if and only if \(det(A) \neq 0\).
  • Let \GL_n(\mathbb{R}) = \{A \in M_n(\mathbb{R}): \text{det}(A) \neq 0\}\), i.e. \(GL_n(\mathbb{R})\) is the set of all invertible \(n \times n\) matrices.
