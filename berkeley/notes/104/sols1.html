  1. Let \(\mathbb{R}\) be the set of real numbers and let the symbols \(<,\leq\) have their conventional meanings.
    a) Show that $$\{x \in \mathbb{R} : 0 \leq x \leq 3\} \cap \{x \in \mathbb{R} : -1 < x < 1\} = \{x \in \mathbb{R} : 0 \leq x < 1\}$$

    Two sets \(X\) and \(Y\) are equal if and only if \(X \subseteq Y\) and \(Y \subseteq X\). Hence we need to show that if \(y\) is an element of the set on the left of the equation it is also an element of the set on the right of the equation, and if \(z\) is an element of the set on the right then it is an elemenet of the set on the left. $$y \in \{x \in \mathbb{R} : 0 \leq x \leq 3\} \cap \{x \in \mathbb{R} : -1 < x < 1\}$$ $$0 \leq y \leq 3 \text{ and } -1 < y < 1$$ $$0 \leq y < 1$$ $$y \in \text{Right Set}$$ $$z \in \{x \in \mathbb{R} : 0 \leq x < 1\}$$ $$0 \leq z < 1$$ $$z \in \{x \in \mathbb{R} : 0 \leq x \leq 3\} \text{ and } z \in \{x \in \mathbb{R} : -1 < x < 1\}$$ $$z \in \text{Left Set } \qquad \square$$

    b) List the elements of $$S = (\{2,3,4\} \cup \{x \in \mathbb{R}:x^2-4x+3=0\})\cap\{x \in \mathbb{R}:-1\leq x < 3\}$$

    The set \(\{x \in \mathbb{R}:x^2-4x+3=0\}\) describes the set \(\{1,3\}\) (you can prove an n-th degree polynomial has at most n roots by factoring out a \(x - root\) and using induction). Therefore: $$S = (\{2,3,4\} \cup \{1,3\})\cap\{x \in \mathbb{R}:-1\leq x < 3\}$$ $$S = \{1,2\} \qquad \square$$

    c) Show that: $$(\{x \in \mathbb{R}:-2\leq x \leq 0\} \cup \{x \in \mathbb{R}:2 < x < 4\})\cap \{x \in \mathbb{R}: 0 \leq x \leq 3\}$$$$ = \{x \in \mathbb{R} : 2 < x \leq 3\}\cup \{0\}$$

    Throwing away a bit of formality and decency, we can argue that: $$\text{LHS} = (\{x \in \mathbb{R}:-2\leq x \leq 0\} \cup \{x \in \mathbb{R}:2 < x < 4\})\cap \{x \in \mathbb{R}: 0 \leq x \leq 3\}$$ $$ = \{ x \in \mathbb{R} : (-2 \leq x \leq 0) \text{ or } (2 < x < 4)\} \cap \{x \in \mathbb{R} : 0 \leq x \leq 3\}$$ $$ = \{x \in \mathbb{R} : x = 0 \text{ or } 2 < x \leq 3\}$$ $$ = \{x \in \mathbb{R} : 2 < x \leq 3\} \cup \{0\}$$ $$ = \text{RHS} \qquad \Box$$
  2. Let \(A, B, C\) be subsets of a set \(S\). Prove the following statements and illustrate them with diagrams.
    a) \(A^\complement \cup B^\complement = (A\cap B)^\complement\)


    All these simple examples seem to boil down to some logical axiom about the relationship between and or and not statements, though that might be just because I haven't learnt set theory yet. $$A^\complement \cup B^\complement = \{x \in S : x \notin A\} \cup \{x \in S: x \notin B\}$$ $$= \{x \in S : x \notin A \text{ or } x \notin B\}$$ $$= \{x \in S : x \notin (A \cap B)\}$$ $$= (A \cap B)^\complement \qquad \Box$$

    b) \(A \cap (B \cup C) = (A\cap B)\cup(A\cap C)\)


    Not much to this one either. $$A\cap (B \cup C) = \{x \in S : x \in A \text{ and } x \in (B \cup C)\}$$ $$= \{x \in S : (x \in A \text{ and } x \in B) \text{ or } (x \in A \text{ and } x \in C)\}$$ $$= \{x \in S : x \in (A \cap B) \text{ or } x \in (A \cap C)\}$$ $$= (A \cap B) \cup (A \cap C) \qquad \Box$$

    c) \(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)


    This one is a bit trickier, let's see why: $$A \cup (B \cap C) = \{x \in S : x \in A \text{ or } x \in (B \cap C)\}$$ $$ = \{x \in S : (x \in A) \text{ or } (x \in B \text{ and } x \in C)\}$$ Now, comparing this set to what we are trying to prove, we see that essentially it boils down to the logical statement that \(A \lor (B \land C)\) is the same as \((A \lor B) \land (A \lor C)\) - (Where \(\land\) is taken to mean 'and' and \(\lor\) is taken to mean 'or').   3. How many subsets are there of the set \(\{1,2,3,...,n\}\)? How many maps of this set into itself? How many maps of this set onto itself?


    To count the number of subsets of the set \(\{1,2,3,...,n\}\), note that in each subset the nuber \(i\) where \(1 \leq i \leq n\) can either be included in the set, or not included in the set. Then, the number of subsets is \(2^n\), because each of the \(n\) elements is either included or not.

    To count the number of maps of the set into itself, (where I'm taking 'into' to mean a one-one map), then we are simply counting the number of permutations of the original set, which is \(n!\). The same for onto.
